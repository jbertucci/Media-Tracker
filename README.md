This project is intended as a solution to silo-ed media tracker apps such as letterboxd, goodreads, etc. 

The things it does: sets up and maintains a local database with an html front-end that allows for search for the various categories of Movies, Books, and Video Games. 
These categories have lists complete with categorization tags and various datapoints depending on the media category.
The movies and video games categories also have a "battle" game feature that allows the user to pit the media against each other in order to rank them.
There are also statistics pages for each media category, splitting and grouping the items in the media category to tell interesting facts.
